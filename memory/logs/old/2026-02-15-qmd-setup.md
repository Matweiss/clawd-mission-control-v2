# qmd Setup - Local Search & Indexing

**Date:** 2026-02-15
**Status:** ✅ Active

## Installation

Installed via npm from GitHub:
```bash
npm install -g https://github.com/tobi/qmd
```

Requires **Bun** runtime (installed at `~/.bun/bin/bun`)

## Search Commands

### BM25 Text Search
```bash
qmd search "onboarding roadmap"
```

### Vector Semantic Search  
```bash
qmd vsearch "customer onboarding process"
```

### Hybrid Search (BM25 + Vector + Rerank)
```bash
qmd query "sales pipeline"
```

### Get Specific Document
```bash
qmd get docs/craftable-onboarding-roadmap.md:1 -l 40
```

## Update Index After Changes

```bash
qmd update
```

## MCP Mode (for AI integration)

```bash
qmd mcp
```

## Environment Setup

Add to PATH (already configured):
```bash
export BUN_INSTALL="$HOME/.bun"
export PATH="$BUN_INSTALL/bin:$HOME/.npm-global/bin:$PATH"
```

## Notes

- Vector embeddings use Ollama at `OLLAMA_URL` (default: `http://localhost:11434`)
- Embeddings not yet generated (run `qmd embed` to create vectors)
- Index lives under `~/.cache/qmd`
- Fast local search without internet dependency
