# Memory: Feb 18, 2026 - Critical Cron Bug Discovered

## Root Cause: Cron Jobs Not Executing

**Discovered at 3:45 PM PST:**
All cron jobs show "finished" with "ok" status but don't actually execute any work.

**The Problem:**
All jobs use `payload.kind: "systemEvent"` which only logs text — it doesn't trigger agent execution.

**Current (broken):**
```json
"payload": {"kind": "systemEvent", "text": "MORNING BRIEFING..."}
"sessionTarget": "main"
```

**Required (working):**
```json
"payload": {"kind": "agentTurn", "message": "MORNING BRIEFING..."}
"sessionTarget": "isolated"
```
